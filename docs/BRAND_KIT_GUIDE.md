# Как сделать свой брендкит

Брендкит — это описание визуального языка твоего продукта: цвета, фон, скругления, акценты. Один раз описал — дальше kalve сам знает, как рендерить карточки в твоём стиле.

## Шаг 1. Скопируй пример

В папке `kalve/brands/` создай файл `mybrand.py` по образцу `saunuole.py`.

```python
# kalve/brands/mybrand.py
from ..brand import BrandKit


MYBRAND = BrandKit(
    name="mybrand",
    product_name="MyBrand",
    product_url="mybrand.com",
    hashtag="mybrand",

    # ... остальные настройки ниже
)
```

## Шаг 2. Определи фон

`background_gradient` — кортеж из трёх элементов: начальный цвет, конечный цвет, угол по CSS-конвенции.

```python
background_gradient=("#FFFFFF", "#F5F5F5", 180),   # почти белый сверху вниз
background_gradient=("#1a1a2e", "#16213e", 135),   # тёмный диагональный
background_gradient=("#FFE5B4", "#FFB6C1", 45),    # пастельный розово-персиковый
```

Если хочешь просто один цвет фона без градиента — укажи один и тот же hex дважды:

```python
background_gradient=("#F5F5F5", "#F5F5F5", 0),
```

## Шаг 3. Определи внутреннюю карточку

По умолчанию внутри градиентного фона рисуется белая карточка со скруглением 48px и отступом 60px от края. Можно поменять:

```python
inner_card_fill=(255, 255, 255, 255),  # RGBA - белая без прозрачности
inner_card_radius=48,                   # скругление в пикселях
inner_card_margin=60,                   # отступ от края канвы
```

Хочешь карточку под цвет бренда — меняй `inner_card_fill`. Хочешь сделать карточку на весь канвас — ставь `inner_card_margin=0`.

## Шаг 4. Определи цвета текста

Четыре уровня иерархии:

```python
text_primary=(17, 24, 39),     # gray-900 - заголовки, основной контент
text_secondary=(55, 65, 81),   # gray-700 - перевод, пояснения
text_muted=(107, 114, 128),    # gray-500 - подпись под бейджем
text_caption=(156, 163, 175),  # gray-400 - номер карточки, мелкое
```

Если используешь Tailwind — бери gray-900/700/500/400. Если своя палитра — подбери четыре оттенка с уменьшающейся насыщенностью.

## Шаг 5. Определи акцентный цвет

`accent` — это цвет для названия продукта и главной ссылки:

```python
accent=(37, 99, 235),   # blue-600
```

## Шаг 6. Определи tier-цвета

Если используешь шаблон `three_tiers`, нужны 3 цветных бейджа:

```python
tiers={
    "tier1": {
        "pill_bg":   (219, 234, 254),  # светлый фон бейджа
        "pill_text": (29, 78, 216),    # тёмный текст бейджа (контрастный)
    },
    "tier2": {
        "pill_bg":   (224, 231, 255),
        "pill_text": (67, 56, 202),
    },
    "tier3": {
        "pill_bg":   (254, 226, 226),
        "pill_text": (185, 28, 28),
    },
}
```

Ключи — любые (`beginner`/`intermediate`/`advanced`, `easy`/`medium`/`hard`, что удобно). Главное, чтобы ключи в JSON-спецификации совпадали с ключами в брендките.

## Шаг 7. Зарегистрируй брендкит

Открой `kalve/brands/__init__.py` и добавь импорт:

```python
from .saunuole import SAUNUOLE
from .mybrand import MYBRAND  # добавляем эту строку

__all__ = ["SAUNUOLE", "MYBRAND"]
```

Если хочешь использовать бренд через CLI — добавь его ещё в `kalve/__main__.py`:

```python
BRAND_REGISTRY = {
    "saunuole": brands.SAUNUOLE,
    "mybrand": brands.MYBRAND,  # добавляем
}
```

## Шаг 8. Проверь

```bash
python -m kalve examples/my_input.json --brand mybrand --output output/
```

Или через Python:

```python
from kalve import generate
from kalve.brands.mybrand import MYBRAND

generate(spec=..., brand=MYBRAND, output_dir="output/")
```

## Советы

**Не ищи цвета с нуля.** Если у тебя уже есть сайт/приложение, открой его в браузере, нажми правой кнопкой → Inspect → и выцеди цвета из CSS-переменных. Экономит час.

**Не выдумывай шрифты.** Используй тот же шрифт, что на твоём сайте. В kalve по умолчанию подключён Inter — если у тебя другой, положи его в `assets/fonts/` и поправь маппинг в `kalve/typography.py`.

**Контраст бейджа.** `pill_bg` должен быть светлым (100-й оттенок Tailwind), `pill_text` тёмным (700-й). Если взять одинаковую насыщенность — текст не прочитается.

**Не используй тени без необходимости.** В дизайн-системе Šaunuolė теней нет совсем, и это выглядит чище. Если у твоего бренда тени есть — добавим в брендкит позже.
